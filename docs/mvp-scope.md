# MVP Scope

## Inputs
- Ticker
- Optional user overrides (assumptions / drivers)

## Outputs
- Normalized historical financials (5Y annual + 12Q)
- Forecast (2Y quarterly + optional 3Y annual)
- 3-statement model (IS/BS/CF) that ties
- Valuation: DCF + multiples
- Predictions: next quarter revenue/EPS range + FY growth/margin ranges
- Audit trail: source links and transformations
- Export: CSV + Excel-like model dump (later native .xlsx)
